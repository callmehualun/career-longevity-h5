# 职场续命力 H5版本

## 📱 项目简介

职场续命力H5版本 - 24道题测出你的职场生存力,找到破局路径

这是职场续命力小程序的H5版本,可以直接在微信、浏览器中打开,方便在私域(微信群、朋友圈)中分享验证。

**体验地址**: 直接打开 `index.html` 文件

---

## ✨ 核心功能

- ✅ **24道专业测评题目** - 覆盖6个维度的职场能力
- ✅ **六维度雷达图** - Canvas绘制,直观展示能力分布
- ✅ **智能路径推荐** - 基于算法推荐3条职业发展路径
- ✅ **完整报告页** - 5000字深度分析(付费9.9元解锁)
- ✅ **本地存储** - 进度自动保存,可随时继续
- ✅ **移动端优化** - 完美适配各种手机屏幕

## 🚀 快速开始

### 方法1: 本地运行

1. 下载项目文件
2. 直接双击打开 `index.html`
3. 或者用浏览器打开: `file:///path/to/career-longevity-h5/index.html`

### 方法2: 本地服务器

```bash
# 进入项目目录
cd career-longevity-h5

# Python 3
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000

# 浏览器访问
open http://localhost:8000
```

### 方法3: 部署到服务器

将整个文件夹上传到服务器,配置好nginx或其他web服务器即可。

## 📊 测评流程

```
首页 → 开始测评 → 24道题(单题单页) → 查看结果 → 付费解锁完整报告
```

### 1. 首页 (index.html)
- Hero区域: 吸引眼球的标题
- 六维度介绍
- 用户价值说明
- CTA按钮: "开始免费测评"

### 2. 测评页 (assessment.html)
- 进度条显示(1/24)
- 单题单页展示
- 4个选项卡片
- 选中后自动进入下一题
- 支持上一题/下一题
- 进度自动保存到localStorage

### 3. 结果页 (result.html)
- 总分大卡片
- 六维度雷达图(Canvas绘制)
- 各维度详细得分
- 付费引导(9.9元解锁完整报告)
- 重新测评 / 分享结果

### 4. 完整报告页 (report.html)
- 需要先"购买"(localStorage标记)
- 整体评估
- 六维度深度分析
- 3条职业路径推荐
- 服务推荐(1v1咨询299元, 简历优化199元)

## 🧮 测评算法

### 六维度评分

| 维度 | 满分 | 题目数 |
|------|------|--------|
| 技能迭代力 | 20分 | 4题 |
| 职业适应力 | 20分 | 4题 |
| 资源整合力 | 20分 | 4题 |
| 心理韧性 | 15分 | 4题 |
| 商业敏锐度 | 15分 | 4题 |
| 行动执行力 | 10分 | 4题 |

**总分**: 100分

### 五级状态划分

| 分数段 | 等级 | 标签 | 颜色 |
|--------|------|------|------|
| 0-30 | DANGER | 职场濒危物种 | 红色 #EF4444 |
| 31-50 | STAGNANT | 职场卡住者 | 橙色 #F59E0B |
| 51-70 | TRANSITION | 职场转型期 | 蓝色 #3B82F6 |
| 71-85 | EVOLVING | 职场进化者 | 绿色 #10B981 |
| 86-100 | MASTER | 职场掌控者 | 绿色 #10B981 |

### 路径推荐算法

基于6维度得分,计算3条路径的适配度:

1. **技能深化路径**: 技能迭代力(30%) + 商业敏锐度(20%) + 行动执行力(15%)
2. **管理转型路径**: 资源整合力(30%) + 心理韧性(25%) + 职业适应力(15%)
3. **独立品牌路径**: 商业敏锐度(30%) + 行动执行力(25%) + 技能迭代力(15%)

## 🐛 常见问题

### Q1: 雷达图不显示?
**A**: 检查浏览器控制台是否有错误,确保Canvas支持。

### Q2: 数据丢失?
**A**: 检查localStorage是否被禁用,或者浏览器隐私模式。

### Q3: 移动端样式错乱?
**A**: 检查viewport meta标签,确保 `user-scalable=no`。

### Q4: 如何清除测评记录?
**A**: 浏览器控制台执行: `localStorage.clear()`
